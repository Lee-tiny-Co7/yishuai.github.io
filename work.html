Git安装

名称 内容
Windows安装 在git官方下载安装包，然后安装
Linux（Ubuntu）安装 打开一个终端窗口，依次输入下面的三个命令。每输入一个，就按回车键
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install git
Mac下安装 首先安装Homebrew
  1. ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  2. brew doctor
  3. brew update
然后安装git
brew install git
参考文献
  1. git官方下载
  2. git安装指南
单词 get（获得），update（更新），upgrade（升级），install（安装），doctor（医生）

Git使用

名称 内容
命令
  1. 克隆：git clone https://github.com/yishuai/yishuai.github.io.git
  2. 更新：git pull
  3. 状态：git status
  4. 提交：git commit -a -m "add source"
  5. 上传：git push origin master
