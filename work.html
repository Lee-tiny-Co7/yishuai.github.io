Git安装

名称 内容
Windows安装 在git官方下载安装包，然后安装
Linux（Ubuntu）安装 打开一个终端窗口，依次输入下面的三个命令。每输入一个，就按回车键
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install git
Mac下安装 首先安装Homebrew
  1. ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  2. brew doctor
  3. brew update
然后安装git
brew install git

B站投稿和视频嵌入

名称 内容
介绍 我爱B站，我们的教学视频和音频当然需要让广大的B站好友也学习一下。下面是制作方法：
  1. 登录B站后，访问 B站创作中心 - 视频投稿投稿页面，上传视频。
  2. 上传视频时，不要怕麻烦，要详细给出视频的各种标签、介绍哦，这样B站才会知道如何把你的视频推荐给想寻找这些视频的人
  3. 记得在视频的介绍中给出你的Github网站链接哦，这样就会很多人来访问你的网站！
  4. 耐心等待审核通过（一小时左右）
  5. 审核通过后，参考 用iframe/embed在网页上嵌入视频/音乐组件 文中介绍方法，获得你的视频的iframe代码，粘贴到我们的Github网站代码中（可以参考下面的示例）
  6. 把文件上传到Github上，然后用浏览器访问你的网站，就可以看到你的B站视频喽！（如果是本机上打开网页，可能看不到。没关系。上传到网站上，就可以看到了。
