学习路线 - Javascript部分

下面的学习路线从浅至深，建议按下面顺序依次进行学习。

在每一部分学习中，建议按以下步骤认真进行：

  1. 学习：看视频，记笔记，做讲解，更新学习之路。这样做的目的是：建立一个知识框架。听课其实是最高效的学习方法，因为老师对教学内容进行了梳理，建立了各内容之间的逻辑联系， 所以，听课是最简单高效的学习方法。 视频以林大为主，因为林大老师的讲解、PPT、示例代码都给得比较细，比较清楚，容易上手。 但林大有些重要内容没有讲，比如弹性盒子、CSS网格，而清华讲了，所以，这几个部分就看清华视频。
  2. 练习：FCC的练习非常好。每一关都会介绍一个小的知识点，然后通过练习掌握。它的内容也很新，很实战。注意：不建议跳过第一步的看视频，而直接做FCC练习，这是 因为我们需要通过视频，建立知识框架，获得总体认识，这有助于我们在做FCC练习时，明白自己在学习之路的哪个位置。
  3. 实战：学习和练习后，我们往往发现：代码能看懂，也有一个模糊的印象，但遇到一个实际问题（比如做一个网站），还是不知道如何做。所以，需要大量练习。这方面，我们通过作业完成。 所以我们要高度重视我们的作业。完成的作业作品，可依次放到我们的作品主页上。

  2. FCC： Bootstrap 注意：可能需要手工在代码最上方粘贴bootstrap css的链接，才会看到效果
实战
  1. 实例分析: 斯坦福机器学习课程网站
    1. 工作1：分析与学习
      1. 访问网站，基于你已经学过的知识，想象它是如何实现的
      2. 在浏览器中查看网页源代码，看是否和你想象的一致。记录下你觉得新颖的内容
      3. 该网页源代码的第18行左右，用到了jquery这个东东，百度一下，这是什么？
      4. 该网页源代码的第29行左右，用到了bootstrap这个东东，百度一下，这是什么？
      5. 该网页源代码的第78行左右，用到了dl，dt，dd这些HTML标签，东东，百度一下，这是什么？它们和我们学过的Table有什么区别？
      6. 该网页源代码的第148行左右，用到了{% for xx in xx %}，你能猜出它的作用吗？
    2. 工作2：复制和修改
      1. 访问该网站的Github代码，观察它的目录和文件组织方式
      2. 用你的账号登陆Github，参考 Fork 一个示例仓库，复制该网站到你的Github账号下
      3. 设置复制过来的网站，在浏览器访问它。是的，现在它变成你的了
      4. 在你的电脑上，打开终端窗口，Git克隆该网站到你的电脑里
      5. 用Sublime Text修改网站标题为《实务学堂写作课》，在自己电脑上打开看效果
      6. 用git commit，push，将修改好的网站上传到Github上，请欧阳老师审阅
  2. 答案在哪呢？：动态折叠百科全书
  3. 我们爱动画：幻灯自动播放
  4. 别乱输入：输入校验
